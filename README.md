# LinkedOut

Tired of seeing BS on LinkedIn that you don't care to see? Well I have a solution for you!

Block the buzzwords, clean your feed.

# Manual Installation

1. Clone this repo 

```bash
git clone https://github.com/NickTheWilder/LinkedOut.git
```
2. In a Firefox browser go to `about:debugging`
3. Click on "This Firefox"
4. Click on "Load Temporary Add-on"
5. Select the `manifest.json` file in the cloned repo
